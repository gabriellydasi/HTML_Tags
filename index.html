HTML

Definição

(Voltar)

HTML (Hypertext Markup Language) é uma linguagem de marcação usada para criar e estruturar documentos na World Wide Web (WWW). Ele define a estrutura e o layout de uma página da web, permitindo a inclusão de texto, imagens, links e outros elementos multimídia. Ademais, a linguagem consiste em uma série de elementos (ou tags) que são interpretados pelos navegadores da web para exibir o conteúdo de forma formatada e organizada. Esses elementos podem ser usados para definir parágrafos, cabeçalhos, listas, formulários e muito mais, proporcionando uma maneira padronizada de apresentar informações na internet.

História

(Voltar)

A história do HTML começa nos anos 1980 e início dos anos 1990, quando Tim Berners-Lee propôs a ideia da World Wide Web no CERN (Organização Europeia para a Pesquisa Nuclear). Tim propôs a ideia de um sistema de hipertexto que permitiria aos pesquisadores compartilhar e acessar documentos facilmente através de links. Anos mais tarde, em 1989, escreveu uma proposta intitulada "Information Management: A Proposal", onde delineava os conceitos de hipertexto e o que se tornaria a World Wide Web (WWW). Ele imaginou um sistema distribuído de hipertexto baseado na internet, no qual documentos poderiam ser vinculados entre si por meio de links. Um ano depois, Berners-Lee desenvolveu o conceito de HTML, definindo as especificações básicas do HTML, que incluíam um conjunto inicial de tags para criar documentos hipertexto. Essas tags permitiam a inclusão de links, imagens e texto formatado. No mesmo ano, fora criado o primeiro navegador web: WorldWibeWed - também conhecido como Nexus. Também criado por Bernes-Lee, o navegador permitia a visualização de páginas web básicas, mas também atuava como editor, permitindo que os usuários criassem conteúdo diretamente na web. Em 1993, Marc Andreessen e Eric Bina desenvolveram o navegador Mosaic na Universidade de Illinois, que foi o primeiro navegador a popularizar a web para o público em geral. Isso levou a um aumento significativo no uso e desenvolvimento de páginas da web, impulsionando ainda mais a adoção do HTML.

A primeira versão oficial do HTML, conhecida como HTML 1.0, foi lançada em 1993. Ela continha um conjunto limitado de tags, como <h1> para cabeçalhos, <p> para parágrafos e <a> para links. Nos anos seguintes, o HTML evoluiu rapidamente com o lançamento de novas versões, cada uma introduzindo recursos adicionais. A versão atual é o HTML5, o qual foi finalizado como a quinta revisão principal da especificação HTML em 2014. Ele trouxe uma série de novos recursos e melhorias, como elementos de mídia, gráficos vetoriais escaláveis (SVG) e suporte a semântica avançada para tornar o conteúdo mais acessível aos motores de busca e dispositivos assistivos.

(Tim Berners-Lee acessando a World Wide Web (WWW))

O que são Tags?

(Voltar)

Cada arquivo HTML contém marcadores, conhecidos como tags, que são expressões delimitadas por parênteses angulares (< e >) e desempenham o papel de instruções de formatação na linguagem. Um elemento HTML consiste em um nome de tag, atributos, valores e possíveis elementos ou texto dentro dele. Os atributos são usados para ajustar as configurações padrão dos elementos, enquanto os valores determinam como esses ajustes são aplicados.

Elementos HTML podem ser categorizados em dois tipos principais com base na presença de conteúdo dentro deles: elementos com filhos e elementos sem filhos.

- Elementos com filhos

  • Esses são elementos que têm outros elementos HTML ou texto dentro deles.
  • Eles agem como contêineres para agrupar e organizar conteúdo.
  • Exemplos incluem <div>, <p>, <ul>, <table>, entre outros.
  • Podem conter qualquer número de elementos filhos, incluindo outros elementos com ou sem filhos.

- Elementos sem filhos

  • Estes são elementos que não têm nenhum conteúdo dentro deles.
  • Eles são usados principalmente para adicionar funcionalidades específicas ou efeitos visuais à página.
  • Exemplos incluem <img>, <hr>, <br>, <input>, entre outros.
  • Eles são chamados de "vazios" ou "autofechados" porque não têm conteúdo entre uma tag de abertura e uma tag de fechamento; eles terminam com uma barra antes do fechamento angular, como </br> ou </img>.

Exemplo:

<p>Este é um exemplo de um elemento HTML simples sem filhos, como o elemento <hr /> que representa uma linha horizontal.</p>


<div>

<p>Este é um exemplo de um elemento HTML com filhos. Aqui está um parágrafo dentro de uma div.</p>

<u>

<li>Item 1</li>

<li>Item 2</li>

<li>Item 3</li>

</u>

</div>

O primeiro elemento é um parágrafo (<p>) que contém um texto explicativo. Este é um exemplo de um elemento HTML sem filhos.

O segundo elemento é uma (<div>) que contém um cabeçalho (<h2>), um parágrafo (<p>) e uma lista não ordenada (<ul>) com três itens de lista (<li>). Este é um exemplo de um elemento HTML com filhos.

Essa distinção é importante porque afeta como os elementos são tratados pelo navegador e como eles afetam a estrutura e o layout da página HTML. Elementos com filhos influenciam a estrutura e o conteúdo da página, enquanto elementos sem filhos geralmente são usados para adicionar funcionalidades específicas ou elementos visuais à página.

Principais Tags

(Voltar)
  • <p>
    • (Voltar)

    Uma das tags mais básicas e amplamente utilizadas para estruturar o conteúdo textual em uma página web. Cada parágrafo definido pela tag <p> é separado visualmente dos parágrafos adjacentes por espaços em branco.

    Exemplo:

    <p>Primeiro parágrafo</p>

    Primeiro parágrafo

  • <h1>, <h2>, <h3> ... <h6>
    • (Voltar)

    São utilizadas para definir cabeçalhos de diferentes níveis de importância em uma página web. Cada número representa um nível de hierarquia, sendo <h1> o cabeçalho mais importante e <h6> o menos importante.

    Exemplo:

    <h1>Título Principal</h1>

    <h2>Subtítulo Importante<h2>

    <h3>Subtítulo Menos Importante<h3>

    Título Principal

    Subtítulo Importante

    Subtítulo Menos Importante

    <h3>, <h4>, <h5> e <h6> são tags destinadas para subtítulos de menor importância sendo utilizadas como subseções dentro de <h1> e <h2>.

  • <small>
    • (Voltar)

    Utilizada para definir texto com um tamanho de fonte menor do que o texto circundante. É frequentemente usada para exibir texto de nota de rodapé, legenda ou informações auxiliares que são menos importantes que o texto principal da página.

    Exemplo:

    Este é um parágrafo de texto com uma <small>nota de rodapé</small> em tamanho menor.

    Este é um parágrafo de texto com uma nota de rodapé em tamanho menor.

  • <strong>
    • (Voltar)

    Utilizada para indicar que o texto dentro dela deve ser enfatizado visualmente, geralmente exibindo-o em negrito. Ela é usada para destacar parte do texto que é de grande importância, ênfase ou relevância.

    Exemplo:

    Este é um parágrafo com um texto <strong>fortemente</strong> enfatizado

    Este é um parágrafo com um texto fortemente enfatizado.

  • <i>
    • (Voltar)

    Utilizada para enfatizar o texto, indicando que ele deve ser exibido em itálico. No entanto, a partir do HTML5, a tag <i> é considerada obsoleta para esse propósito e é recomendado o uso da tag <em>(ênfase) para textos em itálico com significado enfático.

    Exemplo do uso da tag <i>:

    <i>Este texto será exibido em itálico.<i>

    Este texto será exibido em itálico.


    É importante ressaltar que, embora a tag <i> ainda seja suportada pelos navegadores, ela é considerada obsoleta para fins semânticos. Se o texto em itálico tiver um significado semântico específico, é preferível usar a tag <em> para fornecer ênfase ao conteúdo.

    Exemplo de uso da tag <em> para ênfase:

    <em>Este texto enfatizado será exibido em itálico.</em>

    Este texto enfatizado será exibido em itálico.


    No HTML5, a tag <em> é usada para representar ênfase, e os navegadores geralmente renderizam o texto enfatizado em itálico por padrão.

  • <u>
    • (Voltar)

    Utilizada para sublinhar o texto que está dentro dela. Ela é frequentemente usada para indicar que o texto é um link ou para destacar palavras-chave, termos técnicos ou qualquer outra informação que necessite de ênfase visual adicional através do sublinhado.

    Exemplo:

    Este é um exemplo de texto com uma palavra <u>sublinhada</u>.

    Este é um exemplo de texto com uma palavra sublinhada.

  • <mark>
    • (Voltar)

    Utilizada para destacar uma palavra ou frase importante no decorrer de um texto. O texto marcado pela tag normalmente é destacado pela cor amarela.

    Exemplo:

    <mark>Texto importante</mark>

    Texto importante

  • <li>
    • (Voltar)

    Usada para definir um item em uma lista, seja ela ordenada (<ol>) ou não ordenada (<ul>). Essa tag é essencial para estruturar listas de itens em uma página da web.

    Exemplo de lista ordenada:

    <ol>

    <li>Item 1</li>

    <li>Item 2</li>

    <li>Item 3</li>

    </ol>


    1. Item 1
    2. Item 2
    3. Item 3

    Exemplo de lista não ordenada:

    <ul>

    <li>Item 1</li>

    <li>Item 2</li>

    <li>Item 3</li>

    </ul>


    • Item 1
    • Item 2
    • Item 3

  • <ol>
    • (Voltar)

    Usada para criar listas ordenadas, ou seja, listas nas quais cada item é numerado sequencialmente.

    Exemplo:

    <ol>

    <li>Item 1</li>

    <li>Item 2</li>

    <li>Item 3</li>

    </ol>


    1. Item 1
    2. Item 2
    3. Item 3

  • <ul>
    • (Voltar)

    Usada para criar listas não ordenadas, ou seja, listas nas quais os itens não são numerados sequencialmente, mas geralmente são marcados com um bullet (como um ponto ou um círculo).

    Exemplo:

    <ul>

    <li>Item 1</li>

    <li>Item 2</li>

    <li>Item 3</li>

    </ul>


    • Item 1
    • Item 2
    • Item 3

  • <a>
    • (Voltar)

    Usada para criar um hyperlink, ou seja, um link para outra página da web, arquivo, endereço de email, ou qualquer outro URL.

    Exemplo:

    <a>href="https://www.example.com">Visite o site exemplo</a>

    Visite o site exemplo

    Neste exemplo, o texto "Visite o site exemplo" é clicável e, quando clicado, o usuário será redirecionado para o URL especificado no atributo href.

    Também é possível criar links que abram em uma nova aba ou janela do navegador utilizando o atributo target="_blank":

    <a> href="https://www.example.com" target="_blank">Visite o site exemplo<a>

    Visite o site exemplo

    Aqui, o link abrirá em uma nova aba ou janela do navegador quando clicado.

    Além disso, você pode usar imagens como links, envolvendo a tag <img> dentro da tag <a>:

    <a>

    <img> src="https://i.pinimg.com/564x/e1/10/1d/e1101dc5fa86bcfe42b5e6f00d4499ef.jpg" width="200" alt="Descrição da imagem">

    </a>

  • <hr>
    • (Voltar)

    Usada para criar uma linha horizontal, que é frequentemente usada para separar o conteúdo em seções visuais distintas.

    Exemplo:

    Este é um parágrafo.

    <hr>

    Este é outro parágrafo.


    Este é um parágrafo.

    Este é outro parágrafo.

  • <sub>
    • (Voltar)

    Usada para criar texto subscrito, ou seja, texto que é posicionado abaixo da linha de base normal e é geralmente menor em tamanho.

    Exemplo:

    H<sub>2</sub>O

    H2O

  • <sup>
    • (Voltar)

    Usada para criar texto sobrescrito, ou seja, texto que é posicionado acima da linha de base normal e é geralmente menor em tamanho.

    Exemplo:

    x<sup>2</sup>

    x2

  • <blockquote>
    • (Voltar)

    Usada para representar um bloco de texto citado de outra fonte, geralmente identificado com recuo e itálico.

    Exemplo:

    <blockquote>

    <i>Este é um texto citado de outra fonte.</i>

    </blockquote>

    Este é um texto citado de outra fonte.

  • <font>
    • (Voltar)

    A tag <font> era usada para definir o tamanho, a cor e a face da fonte do texto. No entanto, esta tag foi descontinuada no HTML5 e é considerada obsoleta. Em vez disso, recomenda-se o uso de CSS para estilizar o texto.

    Exemplo (não recomendado, pois a tag <font> está obsoleta):

    <font> color="blue" size="3" face="Arial">Este é um texto estilizado com a tag font.</font>

    Este é um texto estilizado com a tag font.

  • <del>
    • (Voltar)

    Usada para representar texto que foi deletado ou removido de uma versão anterior de um documento. O texto dentro da tag <del> geralmente é exibido com uma linha através dele.

    Exemplo:

    O texto <del>removido </del> foi deletado.

    O texto removido foi deletado.

  • <abbr>
    • (Voltar)

    Usada para definir uma abreviação ou um acrônimo e pode opcionalmente fornecer uma explicação do termo abreviado.

    Exemplo:

    O <abbr> title="World Wide Web">WWW</abbr> é uma parte fundamental da Internet.

    O WWW é uma parte fundamental da Internet.
